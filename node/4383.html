Species : Oryzias javanicus

ชื่อไทย: ปลาซิวข้าวสารชวา
ชื่อสามัญ: Java Ricefish
ชื่อวิทยาศาสตร์: Oryzias javanicus (Bleeker, 1854)
ขนาดเมื่อโตเต็มวัย: 2-2.5 เซนติเมตร
การกระจายพันธุ์: ภาคใต้ตามแนวชายฝั่ง
 การจำแนกชนิด: ปลาซิวข้าวสารขนาดกลาง ลำตัวแบนข้างลึกกว่า O. minutillus, O. songkhramensis และ O. mekongensis แต่ไม่แบนข้างและลึกกว่า O. dancena ปลายครีบก้นแตกเป็นเส้นเล็กน้อยแต่ไม่มากเท่า O. dancena ขอบครีบหางเป็นสีส้ม/เหลือง
ถิ่นอาศัย: ในบริเวณแหล่งน้ำกร่อยขนาดเล็ก เช่นแอ่งน้ำในป่าแสมที่ได้รับอิทธิพลจากน้ำขึ้น/ลง
